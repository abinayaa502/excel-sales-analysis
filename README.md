# Excel Sales & Marketing Analysis Dashboard

This project analyses sales data using intermediate Excel skills and demonstrates
dashboard creation with pivot tables, formulas, and charts.

## Project Overview
The dataset contains sales details such as:
- Product
- Category
- Customer
- Region
- Quantity
- Unit Price
- Sales Amount
- Order Date

Using this data, pivot analysis and dashboard visualizations were created.

## Key Features

### Pivot Analysis
- Region-wise sales
- Category-wise sales
- Customer-wise sales
- Monthly sales trends

### Dashboard Highlights
- Total Sales
- Total Orders
- Column chart visualizing sales performance

## Excel Skills Demonstrated
- Pivot Tables
- SUM & COUNTA formulas
- Chart creation
- Basic KPI calculation
- Data filtering and grouping
